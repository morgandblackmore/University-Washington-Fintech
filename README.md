# 6_SF_Housing_Analysis
Challenge 6 of University of Washington FinTech Boot Camp - Analyze San Francisco housing market

---

## Technologies  
Uses Jupyter Lab, Python.  

---  

## Installation Guide  
1. Download this repo.  
2. Install [Jupyter Lab](https://jupyterlab.readthedocs.io/en/stable/getting_started/installation.html) if you don't already have it.  
3. Open san_francisco_housing.ipynb in Jupyter Lab.  

---  

## Usage  
Simply select "Run All Cells" from the Run menu.
